# Python Pong Game

### How to run 
* Install python 
    * Debian/Ubuntu 
        ```
        sudo apt install python
        ```
    * Arch/Manjaro/Garuda 
        ```
        sudo pacman -S python
        ```
* Run the game 
```
git clone https://github.com/BouzoulasDimitrios/pong_python.git
```
```
cd pong_python
```
```
python main.py
```
